
Usage:  make <command>

where command is one of

  help        : show this information

  status      : show current state of all packages

  install     : install all packages

  purge       : purge all packages

  clean       : clean all packages

  fetch       : fetch all packages to pool directory

  <pkg>       : build and install package

  <pkg>-clean : clean package

  <pkg>-purge : purge package
